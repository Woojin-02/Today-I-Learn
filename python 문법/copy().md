# Copy

1. 얕은 복사
```python
d1 = [1, 2, 3]
d2 = d1  # 얕은 복사
print(d1, d2)
d2[1] = 4
print(d1, d2)
```
`결과 : [1, 4, 3] [1, 4, 3]`

얕은 복사 = 주소값 복사이기 때문에 d2만 수정해도 d1이 함께 수정됨

2. 깊은 복사
```python
d1 = [1, 2, 3]
d3 = d1.copy()  # 깊은 복사
print(d1, d3)
d3[1] = 4
print(d1, d3)
```
`결과 : [1, 2, 3] [1, 4, 3]`

깊은 복사 = 데이터 복사이기 때문에 따로 저장공간이 생성됨
